INSTALACIÓN
------------

:: Instalación de Plone ::

- Descarga el instalador para linux desde la siguiente dirección:
	
	http://plone.org/products/plone/releases/3.3.5

- Descomprime el archivo en la carpeta de tu preferencia
- Dentro de la carpeta descomprimida, ejecuta el siguiente comando:

	$ sh install.sh standalone

- Después de la instalación, puedes iniciar una instancia de la siguiente forma:

	$ cd ~/Plone/zinstance/
	$ bin/plonectl start

- Puedes acceder a tu sitio a través de la siguiente dirección:

	http://localhost:8080/
	

:: Instalación del Tema Canaima Aponwao ::

- Ubica tu archivo buildout.cfg en el directorio $INSTANCE_HOME (normalmente ~/Plone/zinstance/).
- Abrelo con tu editor de texto preferido.
- Agrega las siguientes líneas en el apartado [buildout]:

    [buildout]
    ...
    eggs =
        ...
        canaima.aponwaotheme
    ...
    zcml =
        canaima.aponwaotheme

- Reconfigura tu instancia Plone con el siguiente comando dentro de tu directorio $INSTANCE_HOME:

    $ ./bin/buildout
    
- Instala el Producto desde la aplicación "Quick Installer" del ZMI
